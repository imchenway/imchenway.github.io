    4. иҺ·еҸ–2kжҳҫзӨәеҷЁзҡ„DisplayVendorIDе’ҢDisplayProductID#

    • жҲ‘д»¬е…ҲдёҚжҸ’еӨ–з•ҢжҳҫзӨәеҷЁзҡ„иҝһжҺҘзәҝпјҢиҺ·еҸ–macbookиҮӘе·ұеұҸе№•зҡ„IDпјҢз„¶еҗҺеҶҚжҸ’дёҠеӨ–жҺҘжҳҫзӨәеҷЁиҺ·еҸ–еӨ–з•ҢжҳҫзӨәеҷЁзҡ„IDгҖӮ
      еңЁз»Ҳз«Ҝиҫ“е…ҘеҰӮдёӢе‘Ҫд»Ө

    ioreg -l | grep "DisplayVendorID"

    ioreg -l | grep "DisplayProductID"

    еҰӮеӣҫпјҡ

    • жҲ‘з»ҸиҝҮжҳҫзӨәеҷЁзҡ„жӢ”жҸ’е°ұеҸҜд»ҘзӯӣйҖүеҮәеӨ–жҺҘжҳҫзӨәеҷЁзҡ„дёӨдёӘIDгҖӮDisplayVendorIDдёә2513пјҢDisplayProductIDдёә32795
    • жҲ‘д»¬ж–°е»әдёҖдёӘеҗҚеӯ—дёәDisplayVendorID-XXXXзҡ„ж–Үд»¶еӨ№пјҢе…¶дёӯXXXXжҳҜDisplayVendorIDзҡ„16иҝӣеҲ¶е°ҸеҶҷеҚі9d1пјҢеҲҷж–Үд»¶еӨ№еҗҚеӯ—дёәDisplayVendorID-9d1гҖӮз„¶еҗҺеҶҚеҲӣе»әдёҖдёӘз©әзҷҪж–Үд»¶пјҢиҝҷйҮҢдҪ д»¬еҸҜд»ҘзӣҙжҺҘз”ЁжҲ‘зҡ„жЁЎжқҝиҝӣиЎҢдҝ®ж”№.зӮ№жҲ‘дёӢиҪҪ
    • жҲ‘д»¬е°ҶиҝҷдёӘж–Үд»¶е‘ҪеҗҚдёәDisplayProductID-YYYYпјҢе…¶дёӯYYYYеҚіDisplayProductIDзҡ„16иҝӣеҲ¶е°ҸеҶҷеҚі801bгҖӮ

    5. зј–иҫ‘DisplayProductID-YYYYж–Үд»¶#

    • жҲ‘д»¬дҪҝз”ЁPlistEdit ProеҺ»жү“ејҖиҝҷдёӘж–Үд»¶пјҢз„¶еҗҺеңЁDisplayProductIDе’ҢDisplayVendorIDеӨ„еЎ«еҶҷиҝҷдёӨдёӘеҖјзҡ„10иҝӣеҲ¶еҺҹе§ӢеҖјпјҢз„¶еҗҺдёӢйқўжҢүз…§еҰӮдёӢи§„еҲҷеҺ»и®ҫзҪ®еҜ№еә”зҡ„еҲҶиҫЁзҺҮгҖӮ
    • дҫӢеҰӮжҲ‘иҝҷйҮҢиҰҒи®ҫзҪ® 1920 * 1080 hidpi зҡ„и®ҫзҪ®пјҢжҲ‘и®ҫзҪ® 1920 * 1080 е’Ң 3840 * 2160 дёӨз§ҚгҖӮ
      1920зҡ„16иҝӣеҲ¶жҳҜ00000780пјҢ1080зҡ„16иҝӣеҲ¶жҳҜ00000438пјҢеҗҺйқўйңҖиҰҒжӢјжҺҘдёҠ00000001 00200000
    • еҚіпјҡ
      00000780 00000438 00000001 00200000
      3840зҡ„16иҝӣеҲ¶жҳҜ00000F00пјҢ2160зҡ„16иҝӣеҲ¶жҳҜ00000870пјҢеҗҺйқўйңҖиҰҒжӢјжҺҘдёҠ00000001 00200000
      00000F00 00000870 00000001 00200000
    • жҲ‘д»¬е°ҶиҝҷдёӘж•°жҚ®ж·»еҠ еҲ°ж–Үд»¶дёӯеҺ»гҖӮ
