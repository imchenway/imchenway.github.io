    Vector API 概述#

    JDK 16 引入 Incubator 状态的 Vector API（JEP 338），用于在 Java 层进行 SIMD 向量化计算。适合数值处理、加解密、机器学习前置处理。

    性能测试#

    • 使用 JMH 对比普通循环与 Vector API；
    • 确认硬件 SIMD 支持（AVX2/AVX-512）；
    • 关注 -XX:UseAVX 设置。
