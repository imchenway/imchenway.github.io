    Project Panama 简介#

    Project Panama 旨在改进 Java 与本地代码互操作。外部内存 API（JEP 370/383）提供类型安全的内存访问与释放机制，替代 sun.misc.Unsafe。JDK 19 起作为预览特性。

    MemorySegment 与 MemoryLayout#

    • Arena 管理生命周期；
    • MemoryLayout 描述结构体布局；
    • 支持字节序、对齐等特性。
